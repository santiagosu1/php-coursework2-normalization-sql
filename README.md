# PixelForge Studios - Coursework 2

This project is for Coursework 2 on database normalization and SQL.

## Project Overview
The database represents employees working in a video game company called PixelForge Studios.  
The project includes:
- normalization process up to Third Normal Form (3NF)
- SQL schema creation
- sample data insertion
- required SQL queries

## Files Included
- `normalization_process.md` → explains the normalization steps from raw data to 3NF
- `schema.sql` → contains all CREATE TABLE statements
- `data.sql` → contains INSERT INTO statements
- `queries.sql` → contains all required SQL queries
- `exported_database.sql` → exported MySQL database with structure and data

## How to Import into MySQL
1. Open phpMyAdmin or MySQL CLI
2. Create a database or run `schema.sql`
3. Run `data.sql`
4. Run `queries.sql` to test the required queries

## Database Topic
This database is based on job positions in a video game company.
